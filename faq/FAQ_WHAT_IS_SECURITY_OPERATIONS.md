# What is Security Operations?

## Short answer

**Security operations** is the **continuous application of Security Intelligence** to recurring operational decisions—prioritization, coordination, deferral, and escalation—under uncertainty. It is decision support for daily security work, not a runbook library or SIEM configuration guide.

## Explanation

Practitioners often equate security operations with alert triage volume or shift procedures. In the SentriScope encyclopedia, operations means **how teams reason** when applying exposure context, correlated understanding, and integrated intelligence to queues that never empty.

An [operational decision](../glossary/OPERATIONAL_DECISION.md) (`KID-GLS-0040`) is any recurring judgment that uses that intelligence—not a one-time incident command decision.
