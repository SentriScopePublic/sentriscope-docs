# SentriScope public documentation

**Classification:** PUBLIC · TLP:CLEAR

This repository publishes **sanitized, public** SentriScope documentation for
search engines and LLM indexing.

## What this is

- Public product and cybersecurity knowledge documentation
- Maintained as an allowlisted export from the private SentriScope monorepo
  (`docs/public/`)

## What this is not

- Not the SentriScope application source code
- Not internal engineering documentation
- Not a substitute for the product website or authenticated product

## Source of truth

Authoring happens in the private project. Only files that pass fail-closed
guardrails (path allowlist, PUBLIC/TLP:CLEAR metadata, secret scanning) are
synced here.
